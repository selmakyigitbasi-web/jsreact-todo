## ✨ Özellikler / Features

### 🇹🇷 Türkçe

- **Kalıcı Hafıza:** `localStorage` sayesinde sayfayı yenileseniz de görevleriniz silinmez.
- **Hızlı Kullanım:** `form` yapısı sayesinde hem "Ekle" butonuyla hem de **Enter** tuşuyla görev eklenebilir.
- **Mobil Uyumlu:** CSS Media Queries ile her ekran boyutuna (telefon, tablet, PC) uygun tasarım.
- **Durum Takibi:** Görevleri tamamlandı olarak işaretleme veya listeden tamamen silme.

### 🇺🇸 English

- **Persistent Data:** Tasks are saved in `localStorage`, so they stay even if you refresh the page.
- **Fast Input:** Thanks to the `form` structure, you can add tasks usingHem the "Add" button and the **Enter** key.
- **Responsive Design:** Optimized for all screen sizes (mobile, tablet, PC) using CSS Media Queries.
- **Task Management:** Mark tasks as completed or remove them from the list entirely.
